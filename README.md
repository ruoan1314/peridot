# Peridot Cycle Journal

A mobile-first cycle tracking app with:
- Period entry logging
- Symptoms + notes per cycle
- Basic next-period prediction
- PDF export for doctor appointments

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Tech stack

- Next.js (App Router, TypeScript)
- Tailwind CSS
- jsPDF for export
