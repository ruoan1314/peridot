# Peridot Cycle Journal

A mobile-first cycle tracking app with:
- Period entry logging
- Symptoms + notes per cycle
- Basic next-period prediction
- PDF export for doctor appointments

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Tech stack

- Next.js (App Router, TypeScript)
- Tailwind CSS
- jsPDF for export

## Deploy on Vercel

This project is ready for Vercel:

1. Push to GitHub
2. Import the repo on [Vercel](https://vercel.com/new)
3. Deploy
